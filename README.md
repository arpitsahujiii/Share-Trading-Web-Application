# 💹 Share Trading Web Application (Java JSP Project)

## 📌 Overview
This is a web-based Share Trading Application where users can buy and sell shares in a simulated stock market environment.

This project includes both **User and Admin modules** to simulate a real-world trading platform.

---

## 🚀 Features
- User Registration & Login System  
- OTP Verification  
- Buy & Sell Shares  
- Real-time Stock Display  
- Portfolio Management  
- Admin Panel for Stock Management  

---

## 🛠️ Tech Stack
- Java (JSP/Servlet)  
- MySQL  
- HTML, CSS
